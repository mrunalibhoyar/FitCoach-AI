# 🏋️ FitCoach AI – Personalized Fitness Buddy Agent

## Overview

FitCoach AI is an AI-powered Fitness Buddy Agent developed as part of an IBM SkillsBuild internship project using **IBM watsonx Orchestrate**, **IBM Granite**, and **IBM Cloud Lite**.

The project provides personalized fitness guidance by understanding user goals, fitness level, workout preferences, lifestyle, and health limitations. It acts as a virtual fitness coach that helps users create customized workout plans, develop healthy habits, and receive basic nutrition guidance.

---

## Problem Statement

Many people struggle to maintain a healthy lifestyle because of busy schedules, lack of motivation, and limited access to personalized fitness guidance.

Traditional fitness applications usually provide generic workout plans that do not consider individual fitness goals, experience, or personal preferences.

FitCoach AI addresses this challenge by providing intelligent and personalized fitness recommendations through an AI-powered conversational assistant.

## Project Workflow

1. User enters a fitness-related query.
2. The AI agent collects user information such as age, fitness goals, workout preferences, and health limitations.
3. IBM Granite processes the user's request.
4. The agent retrieves relevant information from the uploaded knowledge base.
5. Personalized workout plans and fitness recommendations are generated.
6. The response is presented to the user in natural language.

## Project Note

This project was developed using **IBM watsonx Orchestrate**, a low-code/no-code AI platform.

Therefore, the project does not contain traditional programming source code. The AI agent was created by configuring:

* Agent Instructions
* Knowledge Sources
* AI Behavior
* Guidelines
* IBM Granite Foundation Model

This repository serves as the project documentation and demonstrates the overall implementation of the Fitness Buddy Agent.
